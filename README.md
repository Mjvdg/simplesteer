# Simplesteer

## Description
Autosteer software with a web interface. 


## Hardware
 - 2 u-Blox ZED-F9P receivers
 - Raspberry pi 3
 - Arduino nano

This repository contains 2 projects. For more info consult their readmes.
    
- [frontend](/frontend/)
- [backend](/backend/)

## Features

- ab-lines
- PWM steering (elektric motor)
- Pulses steering ()
- ublox AssistNow
- saving of lines
